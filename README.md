# CFTR QSAR Tool

**Easy tool to predict CFTR potentiator activity**

Open-source QSAR pipeline for medicinal chemists working on **CFTR modulators** (cystic fibrosis therapies from Verkman lab).

It learns from 13 CFTR primary hits and gives quick predictions of fold stimulation.

### How to use it (3 simple steps)

1. Click the button below:

   [![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/github/drjoykarmakar/cftr-qsar/blob/main/cftr-qsar.ipynb)

2. Click **Runtime → Run all**

3. You will see the model results and scaffold analysis.

### Important note
This is an early version with 13 compounds. Predictions will improve significantly once more diverse CFTR compounds are added.
